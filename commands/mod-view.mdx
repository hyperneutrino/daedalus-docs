---
title: "/mod-view"
---

<Note>By default, this command requires the **Moderate Members** permission.</Note>

Open Daedalus mod view for a user. This shows their ID, avatar, banner, mod action count, and ban, mute, and timeout status. Depending on your permissions,
it will also show controls for banning/unbanning, muting/unmuting, and timing out/removing their timeout for them. The default instant-timeout duration is
configured via the dashboard.

## Syntax

```
/mod-view <user>
```

| Parameter | Information       |
| --------- | ----------------- |
| `user`    | The user to show. |

## Context Menu

This command is also available as a context menu command -- right click / long press a user and select **Apps > Mod View**.
