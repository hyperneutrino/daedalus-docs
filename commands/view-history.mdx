---
title: "/view-history"
---

<Note>By default, this command requires the **Moderate Members** permission.</Note>

View a user's mod action history.

## Syntax

```
/view-history <user> [show-hidden]
```

| Parameter     | Information                                |
| ------------- | ------------------------------------------ |
| `user`        | The user to show.                          |
| `show-hidden` | If true, also show hidden history entries. |

## Context Menu

This command is also available as a context menu command -- right click / long press a user and select **Apps > View History**. `show-hidden` will
automatically be `false`.
