---
title: "/massban import"
---

<Note>
    By default, this command requires the **Manage Server** and **Ban Members** permissions. This command requires the bot to have the **Ban Members**
    permission.
</Note>

Massban users by providing an export key (produced by `/massban export`). If configured, this requires either a set number of approvals or someone with a
bypass role to skip approval. Users will not be notified.

## Syntax

```
/massban import <key> [reason] [purge-duration]
```

| Parameter        | Information                                                                             |
| ---------------- | --------------------------------------------------------------------------------------- |
| `key`            | A Daedalus massban export key.                                                          |
| `reason`         | The reason to log.                                                                      |
| `purge-duration` | Purge the specified period of chat history from this user if applicable (up to 7 days). |
