---
title: "/xp reset-user"
---

<Note>By default, this command requires the **Administrator** permission.</Note>

Reset a user's XP. This will set their XP back to 0 (including their monthly, weekly, and daily XP gains). Their roles will be updated according to your reward
role configuration (including restoring roles that are removed at certain levels). You will be prompted to confirm this action.

## Syntax

```
/xp reset-user <user>
```

| Parameters | Information                    |
| ---------- | ------------------------------ |
| `user`     | The user for whom to reset XP. |
