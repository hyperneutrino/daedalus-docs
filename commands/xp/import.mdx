---
title: "/xp import"
---

<Note>By default, this command requires the **Administrator** permission.</Note>

Import XP from another bot. After running this command, you will be prompted to select the import mode. These modes are explained in the prompt. After
selecting the import mode, you will be asked to confirm one last time. This action cannot be done after that confirmation.

## Syntax

```
/xp import <source>
```

| Parameter | Information                      |
| --------- | -------------------------------- |
| `source`  | The source from which to import. |
