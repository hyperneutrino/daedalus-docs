---
title: "/slowmode"
---

<Note>By default, this command requires the **Bypass Slowmode** permission. This command requires the bot to have the **Manage Channels** permission.</Note>

Set the current channel's slowmode (ratelimit per user).

## Syntax

```
/slowmode <delay>
```

| Parameters | Information                                                                                                 |
| ---------- | ----------------------------------------------------------------------------------------------------------- |
| `delay`    | The new slowmode (delay between consecutive messages) (up to 6 hours). If 0, remove the channel's slowmode. |
