---
title: "/note"
---

<Note>By default, this command requires the **Moderate Members** permission.</Note>

Add a private mod note to a user. The user will not be notified.

## Syntax

```
/note <user> <private-notes>
```

| Parameter       | Information              |
| --------------- | ------------------------ |
| `user`          | The user to warn.        |
| `private-notes` | A private reason to log. |
